# Personal .dotfiles

## Mac

- git
- nvim
- zsh

## Ubuntu

- nvim
- bash

## Arch Linux

- alacritty
- AwesomeWM
- nvim

These are my current dotfiles for either Mac or Linux.
To setup on a Mac:
```
# install stow
brew install stow

# run install script
source ./mac/install.sh

# run git setup script
source ./mac/git_setup.sh
```

To setup on Ubuntu:
```
# install stow
sudo apt install stow

# run install script
source ./linux/install.sh

# run git setup script
source ./linux/git_setup.sh
```
